# WT-RetroWave
Windows Terminal Retrowave Based theme

background: `#0F0C29`

foreground: `#F8F8F2`

cursorColor: `#FFB86C`

## Installation
Replace ```settings.json``` on ```%userprofile%\AppData\Local\Packages\Microsoft.WindowsTerminal_8wekyb3d8bbwe\LocalState```
